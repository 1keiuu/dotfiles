# dotfiles

## basic setup
```
$ sh scripts/mac.sh
$ sh symlinks.sh
$ init.sh
```

## additional setup

### iTerm
- load iTerm setting file(`iTerm.json`) from `iTerm2 > Preferences > General > Preferences`

## about scripts
#### `mac.sh`
  setup macOS preferences
#### `symlinks.sh`
  create symbolic links in $HOME
#### `init.sh`
  install and bundle brew packages
